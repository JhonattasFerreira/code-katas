# The Nature of Code

Exercises from [The Nature of Code](https://natureofcode.com/) by [Daniel Shiffman](https://github.com/shiffman).

Unlike algorithm puzzle sites, these exercises focus on simulating natural phenomena — autonomous agents, physics systems, noise-driven movement, and more.
