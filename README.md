# Code Katas

A unified repository where I store coding challenges I work through to sharpen my skills.

## Structure

Each folder at the root represents a different challenge platform. Navigate into it for the full list of challenges.

```text
code-katas/
└── codingchallenges/   # https://codingchallenges.fyi/
    └── ccwc/
```

## Platforms

| Platform | Description |
| -------- | ----------- |
| [Coding Challenges](codingchallenges/) | Real-world tools built from scratch — CLI utilities, servers, databases, and more |
